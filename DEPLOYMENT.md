# Инструкция по развертыванию в продакшене

## Подготовка к деплою

1. **Проверьте конфигурацию локально:**
   ```bash
   npm run check-config
   ```

2. **Убедитесь, что все изменения закоммичены:**
   ```bash
   git add .
   git commit -m "Обновлена модель AI на GPT-5"
   git push origin main
   ```

## Переменные окружения для продакшена

В продакшене необходимо установить следующие переменные окружения:

### Обязательные:
- `OPENAI_API_KEY` - ваш API ключ от OpenAI
- `OPENAI_MODEL` - модель для использования (например, `gpt-5`)

### Рекомендуемые:
- `NODE_ENV` = `production`
- `PORT` = `10000` (или другой доступный порт)

## Деплой на Render

1. В Dashboard Render перейдите в настройки вашего Web Service
2. В разделе "Environment Variables" добавьте:
   - `OPENAI_MODEL` = `gpt-5`
3. Перезапустите сервис

## Проверка после деплоя

1. Откройте ваше приложение
2. Попробуйте загрузить фото блюда
3. Проверьте логи в Render Dashboard на наличие ошибок

## Откат изменений

Если возникнут проблемы, можно быстро откатиться:
```bash
git revert HEAD
git push origin main
```

## Поддержка

При возникновении проблем проверьте:
- Логи в Render Dashboard
- Консоль браузера
- Статус OpenAI API
